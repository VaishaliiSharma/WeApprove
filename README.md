# **WeApprove - Loan Approval Prediction System**
**Project Overview -**

WeApprove is an AI-powered loan approval prediction system that uses AIML chatbots to assist users. The system allows users to input their financial details and receive instant predictions on loan approvals based on machine learning models.


**1️⃣ Tech Stack:**

Frontend: React.js

Backend: Node.js, Express.js

Database: MongoDB

Machine Learning: Python (Flask API for ML Model)

Chatbot: Chatbot integration


**2️⃣ Features:**

✔️ AI-based loan approval prediction
 
✔️ AIML chatbot for user assistance
 
✔️ Login authentication of user
 
✔️ User-friendly dashboard to manage loan applications
 
✔️ Secure login and data storage


**3️⃣ Installation & Setup-**

Follow these steps to set up the project locally:

bash

**Clone the repository**

git clone <https://github.com/VaishaliiSharma/WeApprove>

cd WeApprove

**Install backend dependencies**

npm install

**Start the backend server**

node server.js

**Install frontend dependencies**

npm install

**Start the frontend server**

npm start


**Start the ML model API**

cd ml-model

python app.py


**4️⃣ Usage:**

✔️ Log in or sign up.

✔️ Enter financial details and request a loan approval prediction.

✔️ Chat with the chatbot for assistance.

✔️ Get an instant loan approval prediction.
